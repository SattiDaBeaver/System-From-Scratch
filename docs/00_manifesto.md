# System from Scratch Manifesto

System from Scratch is a minimalist 32-bit RISCV compatible CPU along with a custom OS designed for clarity, extensibility, and long-term architectural exploration.

## Design Goals
What's in scope for the project:
- Fully documented microarchitecture
- Clean modular RTL
- Support for privilege modes
- Future MMU support
- Custom OS

Some high-reach goals for future me:
- Custom programming language and compiler
- Write an OS in the custom language
- Add MMU and cache

What is not in scope:
- Out-of-order execution
- Superscalar execution
- High-performance optimization in v1
